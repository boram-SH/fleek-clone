SUMMARY:
IMPORTANT FINDING ABOUT THE "WEB VERSION": Fleek has no functional web app. https://app.fleek.fit/en is a minimal Next.js landing page containing only the tagline "Intuitive Workout Tracker", a dumbbell icon, Google Play / App Store download buttons, and a footer (Notice, Privacy, ToS, mailto:support@fleek.fit, "(c) 2025 DLT Partners Inc."). Probing routes (/en/login, /en/trainer, /en/stats, etc.) all return 404; only /en, /en/notice, /en/legal/* exist. The /notice board contains only administrative posts (Sept 2025: an urgent iOS data-loss incident + fix, Apple Sign-in delays, and business-transfer/privacy-policy changes — the company was transferred, consistent with the MWM acquisition). So the entire product surface is the mobile app (Android package com.primero.fleekfrontend, iOS id 1576993198, by DLT Partners Inc., Seoul; EN/KO localized; v3.7.40 as of July 2026; ~185MB iOS; wearOS + Apple Watch companions). This is good news for an offline local rebuild: there is no web parity to match, and reviews confirm the real app is server-dependent (does not work offline; login/data-loss incidents), which is a weakness the rebuild can fix.

SCREENS (reconstructed from the 8 official App Store screenshots, store descriptions, and reviews): (1) RECORD tab — a calendar screen titled "Record" with three sub-tabs: Workouts / Photos / Body. Month-grid calendar with colored dot markers under each date (orange = workouts, yellow = photos, white = body entries) and a legend; below the calendar, a list of sessions for the selected day (e.g., "Push & Pull — 2023-12-25 16:09, 60 minutes") with a thumbs-up icon and an overflow menu; a "+" button top-right to add an entry. (2) ROUTINE screens — routine detail (e.g., "Push & Pull") shows two anatomical figures (front + back view) with the routine's target muscles shaded red, followed by the exercise list: each row has an illustration thumbnail, name incl. aliases ("Barbell Shoulder Press (Overhead Press, Military Press)"), muscle tags ("Shoulder, Triceps"), and a per-exercise "..." menu. Free tier allows 4 routines. (3) LOGGING screen — top bar: back arrow, elapsed session timer (e.g., 00:32), a row of progress dots, "..." menu. Below: animated exercise illustration (3D-mannequin GIF with active muscles in red), exercise name, collapsible "Memo" section (per-exercise note), and a "Record" table with columns Set | KG | Reps | Done(checkbox) and a per-set "..." menu (set-type options; reviews confirm drop sets, supersets, and failure sets are supported). Footer actions: "+ Add Set", "- Remove Set", "All Done" (green), "Next Exercise" (arrow). A rest timer runs between sets with an alarm; rest duration is configurable globally and per routine (default 60s); expert-recommended rest times are auto-suggested. A "substitute exercise" function swaps an exercise mid-workout. Dumbbell exercises automatically count weight x2 in volume; some machine exercises include bodyweight in volume (a review complains this inflates leg-press-machine calf-raise volume). Rest timer only works reliably with the app foregrounded (complaint). (4) WORKOUT SUMMARY screen — routine name + date, front/back muscle figures shaded by what was worked, then a stat grid: WORKOUT (ordinal count, "331st"), CALORIES (304 KCAL), DURATION (60 min), VOLUME (7,735 kg), EXERCISES (8), SETS (24), REPS (337), INTENSITY (128 kg/min = volume/duration). Below, a per-exercise breakdown: "Barbell Shoulder Press — MAX Weight: 35kg | 1RM: 45kg — Top 44%" plus set chips showing weight x reps per set (20x12, 30x10, 30x10, 30x10, 35x8). Summary is shareable as text/image. (5) EXERCISE DETAIL — two tabs: "Instructions" (sub-tabs: Tips / Instructions / Precautions; animated illustration; muscle tags; prose with key phrases highlighted in red) and "My Records" (sub-tabs: Top Records / Recent Records / Chart / Top Percentage). Top Records cards: 1RM MAX (67.37 kg), VOLUME MAX (658.68 kg), WEIGHT MAX (62.5 kg). Recent Records: date-grouped set lists (per-set weight/reps). Chart: per-exercise progress graph over time. Top Percentage: percentile benchmarking against a cohort ("23~39 Men, 70kg weight user's 1RM performance — My 1RM is 67.4kg, top 11.2%") drawn as a distribution curve with a dashed line at the user's level. (6) STATS tab — titled "Stats": "Muscle Fatigue" section with front/back anatomical heatmap figures colored by current fatigue, a caption of the contributing session ("Push & Pull, 12/25/2023 16") and a horizontal time slider to scrub the fatigue state through history; below it "Used Muscle" with period tabs 7 days / 30 days / 90 days / Total Period. Reviews mention a Weekly Summary chart and 1-Month Tendency chart that are Pro-gated. (7) SOCIAL — follow users, view their workout records, upload/share routines, copy routines from others; relative comparison of performance with other users. (8) TRAINER MODE — trainers manage multiple members, prescribe/edit routines per member, review members' logs and give feedback (recent updates repeatedly "improve trainer mode"). (9) AI/EXTRA (marketing-era additions, non-core): AI-recommended routines, InBody (body composition) linking, entering your gym so routines use its equipment, celebrity/bodybuilder routines, expert tips on form/nutrition/recovery.

EXERCISE LIBRARY: 400+ exercises organized by body part/muscle group and workout category (strength: barbell/machine/free-weight; cardio: treadmill, cycle, rowing; bodyweight/home training; band), each with an animated mannequin GIF, target-muscle diagram, aliases, tips/instructions/precautions text. Users can create custom exercises, but (review complaint) custom exercises only appear in the diary — they get no weight/reps/1RM charts, and no stopwatch-type custom logging (plank-style) exists. Logging modes are per-type: weight x reps for strength, duration/distance for cardio.

MONETIZATION (for scoping the free-tier UX): Fleek Pro at $2.99/mo, $23.99/yr, $69.99 lifetime (KR: W3,900/W29,000/W89,000). Free tier: max 4 routines, Pro-gated summary/tendency charts, ads added in recent versions. KNOWN WEAKNESSES worth fixing in a rebuild: online-only (no offline mode), account/login required with re-login annoyances, Sept 2025 server-side data loss, rest timer needs foreground, saved records hard to edit after completion (must delete and re-enter, per one review), custom exercises excluded from stats, real-time calorie count not shown during workout, no per-set differentiated pre-set rest times, no per-exercise elapsed-time breakdown, exercise substitution list is slow to browse mid-workout.

Also has a home-screen widget showing workout days.

Used to decide what to train today.
